# Project-S3
Clarity Map

A React-based dashboard application visualizing pollution zones and environmental mission opportunities in Yerevan, Armenia.

This project is designed to help citizens and environmental groups identify high-priority areas for cleanup and track progress toward reducing pollution.

## Features

* **Pollution Map:** Interactive map showing pollution hot spots with location-specific data.
* **Severity Markers:** Zones are marked by size and color based on severity (High, Medium, Low).
* **Filtering:** Users can filter visible zones by severity level.
* **Detailed Zone Info:** Clicking a zone displays its name, ID, severity, and mission reward points in the sidebar.
* **Real-time Stats:** Overview of total zones per severity level and key project metrics (Active Users, Pollution Trend, Total Points).
* **Technology Stack:** Built with modern web technologies for a fast and responsive experience.
